# Denticollis linearis example

This folder is a lightweight, sanitized example of the sort of output produced for `PRJEB59796` / `icDenLine1`.

It intentionally omits:

- machine-local bibliography and CSL paths
- internal storage paths from YAML sidecar files
- large generated image files

The committed files here are enough to show the expected output shape without exposing environment-specific details.
