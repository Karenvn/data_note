# Example output folders

This directory contains lightweight representative output folders for discussion and orientation.

They are not intended to be exact golden-master outputs because real runs can include:

- machine-local template paths
- internal filesystem paths
- environment-specific JIRA or portal enrichment
- large generated figure files

Use these folders to show colleagues the expected folder shape and core text artifacts:

- `<species>/<tolid>.md`
- `<species>/<bioproject>_context.csv`
- optional figures such as `Fig_2_Gscope.gif` and `Fig_5_Snail.gif`
